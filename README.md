# Bibliopolium

Chrome extension starter project.

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click "Load unpacked" and select this folder.

## Structure

- `manifest.json` - Extension manifest (MV3).
- `popup.html` / `popup.css` / `popup.js` - Toolbar popup UI with storage example.
- `background.js` - Service worker entry.
- `content.js` - Content script stub.
